# CommandMenu 组件示例任务

## 任务概述

创建 CommandMenu SwiftUI 组件的完整示例包，展示 CommandMenu 和 CommandGroup 的各种用法。

## 实现计划

### 1. 基础示例

- CommandMenu 基本用法
- 简单的菜单项和分隔符
- 键盘快捷键绑定

### 2. CommandGroup 示例

- 在现有菜单组中添加命令
- 在菜单组前后插入命令
- 替换现有命令组

### 3. 高级示例

- 嵌套菜单结构
- 动态菜单内容
- 条件菜单项
- 自定义菜单项样式

### 4. 实际应用场景

- 文件操作菜单
- 编辑操作菜单
- 视图控制菜单
- 帮助和工具菜单

## 技术要点

- CommandMenu 在 macOS 上显示为菜单栏菜单
- 在 iOS/iPadOS 上创建键盘快捷键
- 与 Scene 的 commands 修饰符配合使用
- CommandGroup 用于扩展现有菜单

## 平台支持

- iOS 14.0+
- iPadOS 14.0+
- macOS 11.0+
- visionOS 1.0+

## 完成标准

- 所有示例在 iOS 和 macOS 平台编译通过
- 包含详细的中文注释
- 提供完整的文档说明
- 涵盖所有主要用法和修饰符
